## What is Glint 247?

Glint 247 is a professional networking platform designed for creative minds. Whether you're a designer, developer, writer, or any other creative professional, Glint 247 helps you connect with peers, discover opportunities, and grow your career.

<Info>
  Glint 247 is built with Next.js 14, React 18, and Tailwind CSS — making it fast, modern, and easy to extend.
</Info>

## Key capabilities

- **Secure authentication** — JWT-based login and registration with bcrypt password hashing
- **Guided onboarding** — Multi-step flow that personalizes your profile right after sign-up
- **Protected routes** — Automatic redirect to login for unauthenticated users
- **Interactive landing page** — Swipeable carousel showcasing the platform with calls to action
- **Responsive design** — Fully mobile-friendly layout powered by Tailwind CSS

## Technology stack

| Layer | Technology |
|---|---|
| Framework | Next.js 14 (App Router) |
| UI Library | React 18 |
| Styling | Tailwind CSS |
| Authentication | JWT (jsonwebtoken) + bcryptjs |
| HTTP Client | Axios |
| Carousel | Swiper + react-swipeable |
